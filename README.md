##  Description
Ce projet porte sur le **traitement d’image** à travers une interface graphique développée sous **Scilab**.  
L’objectif est de proposer une interface simple et intuitive permettant de manipuler, transformer et analyser des images numériques.

##  Fonctionnalités
- Chargement et affichage d’images dans l’interface
- Application de filtres et transformations (contraste, luminosité, niveaux de gris, etc.)
- Détection et segmentation de zones d’intérêt
- Visualisation des résultats directement dans la fenêtre graphique
- Organisation du code en modules pour une meilleure lisibilité et réutilisabilité

##  Technologies utilisées
- **Scilab** (environnement scientifique et numérique)
- **Toolbox GUI de Scilab** pour la création d’interfaces
- Fonctions de traitement d’image intégrées et personnalisées
